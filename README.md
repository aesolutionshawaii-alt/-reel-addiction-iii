# Reel Addiction III

O'ahu's Premier Deep Sea Fishing Charter - Website Redesign

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Hosting:** Vercel
- **Booking:** FareHarbor (embedded widget)

## Design System

### Colors
- Navy: `#0c1e3c`
- Off-white: `#f7f5f2`
- Red (accent): `#c41e3a`
- Gold (stars): `#f4c542`

### Typography
- **Headings:** Outfit (Google Fonts)
- **Body:** Inter (Google Fonts)

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Project Structure

```
src/
├── app/
│   ├── globals.css      # Tailwind + custom styles
│   ├── layout.tsx       # Root layout with metadata
│   └── page.tsx         # Homepage
├── components/
│   ├── Hero.tsx         # Hero section with nav
│   ├── ExploreCards.tsx # Vessel, Experience, Catch cards
│   ├── CharterSection.tsx # Charter options
│   ├── Testimonial.tsx  # Google review quote
│   ├── SeaKeeperSection.tsx # SeaKeeper callout
│   ├── FinalCTA.tsx     # Final booking CTA
│   └── Footer.tsx       # Site footer
└── public/
    └── images/          # All site images
```

## Image Requirements

Add these images to `/public/images/`:

- `hero-boat.jpg` - Wide boat shot for hero (1920x1080 recommended)
- `logo-white.png` - White logo for dark backgrounds
- `logo-dark.png` - Dark logo for light backgrounds
- `vessel.jpg` - Boat profile shot
- `experience.jpg` - Action shot (fishing)
- `catch.jpg` - Group with fish haul
- `charter-34day.jpg` - 3/4 day charter image
- `charter-fullday.jpg` - Full day charter image
- `charter-extravaganza.jpg` - Extravaganza charter image
- `charter-custom.jpg` - Custom trip image
- `cta-group.jpg` - Group photo for final CTA
- `seakeeper-logo.png` - SeaKeeper logo

Deployed automatically to Vercel on push to `main` branch.
